# Music-Player
 
## Technologies used :
###  1) JAVA
###  2) SWING(jFrame)
###  3) jLayer

## Functionality :
###  => Play, Pause, Reset, Next and Previous buttons. 
###  => Can add multiple songs to playlist
###  => Have rich animation for each song
